# Baikal-Amikan
## React + TypeScript + Vite

To modify the app locally 
```bash
cd app 
npm i && npm run dev
```

To deploy new version of the app
```bash
cd app
npm run build && npm run preview  # to check the build

# commit the changes and push to the repo
git add .
git commit -am "version 1.2.1"
git push
```
